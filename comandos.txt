comandos conda
conda update (nome pacote)  = Para atualizar um pacote
conda create -n (Nome env) python=x.x(Versão do Python) = Para criar um env
conda install -n yourenvname [package] = Para instalar um pacote no env atualizar
conda remove -n yourenvname -all = Para excluir um ambiente pacote do ambiente conda
conda env remove -n ENV_NAME = Para exlcuir um ambiente conda
conda env list = Lista todos ambientes virtuais
conda activate (Nome ambiente virtual) 
conda deactivate
